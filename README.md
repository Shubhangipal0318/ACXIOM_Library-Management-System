# Library Management System

A web-based Library Management System built using HTML, CSS, and JavaScript.
Developed as part of a technical assignment for the Trainee Technical Consultant role.

## 🔐 Login Credentials
| Role  | Username | Password |
|-------|----------|----------|
| Admin | adm      | adm      |
| User  | user     | user     |

## ✨ Features
- Role-based login (Admin & User)
- Book & Movie Management
- Member Management
- Issue / Return / Pay Fine Transactions
- Reports (Active Issues, Overdue Returns, Memberships & more)
- Maintenance Module (Admin only)
- Form Validations on all pages
- Fully responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Run
1. Clone or download the repository
2. Open `index.html` in any browser
3. No installation or server required

## 📁 File Structure
```
📁 library-management-system/
├── index.html   → Structure & screens
├── style.css    → Styling (Cobalt Blue theme #3D5A80)
└── app.js       → All logic & page rendering
```

## 📌 Notes / Assumptions
- Fine is calculated at ₹2 per day overdue
- Data is stored in-memory (resets on page refresh)
- Author field added to Add Book form for better usability
